# Bioanalytical Stage 1 Manifest Import - Example CSV

## Purpose

This example CSV demonstrates how to import samples for **STAGE 1: Sample
Reception & Registration** of the bioanalytical workflow.

## Stage 1 Requirements Coverage

This CSV covers all 5 Stage 1 requirements:

### 1. ✅ Biological Samples from Medical Laboratory

- **BIO-2024-001 to BIO-2024-007**: Plasma, Serum, Urine, Whole Blood samples
- **Source**: Medical Laboratory - Clinical Site A/B/C
- **Clinical site linkage**: Captured in Source Origin field

### 2. ✅ Pharmaceutical Samples from Researchers/External Clients

- **API-2024-001**: Active Pharmaceutical Ingredient
- **TAB-2024-001**: Tablet formulation
- **CAP-2024-001**: Capsule formulation
- **SUS-2024-001**: Suspension formulation
- **REF-2024-001**: Reference Standard
- **Source**: External Client, Researcher, CRO Partner

### 3. ✅ Sample Identifier with Source Linkage

- **Sample ID**: Unique identifier (BIO, API, TAB, CAP, etc.)
- **Source tracking**: Maintained through Source Origin field
- **Auto-generated**: Accession Number and Barcode will be generated by system

### 4. ✅ Metadata Registration

- **Sample Type**: Validated against bioanalytical lab types
- **Requested Tests**: LC-MS/MS, HPLC, Assay, Dissolution, Bioequivalence
- **Storage Conditions**: Room Temperature, Refrigerated, Frozen (-20°C, -80°C)
- **Source**: Medical Laboratory, External Client, Researcher

### 5. ✅ Study Linking

- **Project/Study Association**: BE-2024-001, STAB-API-001, PK-STUDY-002, etc.
- **Subject/Patient ID**: SUBJ-001 to SUBJ-005 (for bioequivalence studies)
- **Timepoint**: Pre-dose, 1h, 2h, 4h, 8h (for pharmacokinetic studies)

## Sample Types Included

### Biological Samples (from Medical Laboratory)

- **Plasma**: For bioequivalence studies
- **Serum**: For bioequivalence studies
- **Urine**: For pharmacokinetic studies
- **Whole Blood**: For special analyses

### Pharmaceutical Samples (from Researchers/External Clients)

- **API**: Active Pharmaceutical Ingredients
- **Tablet**: Finished pharmaceutical products
- **Capsule**: Encapsulated formulations
- **Suspension**: Liquid formulations
- **Reference Standard**: Quality control standards

## Test Data Scenarios

1. **Bioequivalence Study (BE-2024-001)**

   - Multiple subjects (SUBJ-001 to SUBJ-004)
   - Multiple timepoints (Pre-dose, 1h, 4h, 8h)
   - Plasma and serum samples
   - LC-MS/MS and bioequivalence testing

2. **Stability Study (STAB-API-001)**

   - API and finished products
   - Assay and dissolution testing
   - Room temperature storage

3. **Pharmacokinetic Studies (PK-STUDY-002, PK-STUDY-003)**

   - Various biological matrices
   - Time-course sampling
   - LC-MS/MS analysis

4. **Formulation Development (FORMDEV-001, FORMDEV-002)**
   - Prototype formulations
   - Researcher samples
   - Development testing

## How to Use

1. **Download** the example CSV file:
   `bioanalytical-stage1-manifest-example.csv`
2. **Navigate** to the bioanalytical workflow Stage 1 page
3. **Click** the "Manifest Import" tab
4. **Click** "Import Stage 1 Manifest" button
5. **Upload** the CSV file
6. **Review** the auto-mapped columns in Step 2
7. **Preview** the data in Step 3
8. **Complete** the import in Step 4
9. **View results** in the "Received Samples" tab
